# WeirdHTA


A python tool to create obfuscated HTA script.

```
git clone https://github.com/felamos/weirdhta
sudo apt install npm nodejs
npm i javascript-obfuscator
node run.js
```

```
node run.js 
[+] Usage: node /opt/weirdhta/run.js "CMD"
```
