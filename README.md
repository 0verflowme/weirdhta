# WeirdHTA


A python tool to create obfuscated HTA script.

```
git clone https://github.com/felamos/weirdhta
chmod +x weirdhta.py
./weirdhta.py
```

Restful API : https://enigmatic-shore-46592.herokuapp.com/api/weirdhta
